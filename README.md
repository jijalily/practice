# Определение занятости парковочного места

Прикладная система компьютерного зрения, которая по изображению парковочного
места определяет, свободно оно или занято (бинарная классификация).
Проект реализует полный цикл: подготовку данных, обучение и сравнение
**5 архитектур** нейронных сетей, выбор лучшей модели и её интеграцию в
демонстрационное приложение с визуализацией результатов.

Проект выполнен в рамках производственной практики
«Системы искусственного интеллекта» (направление 09.03.01).

---

## Постановка задачи

Разработать систему бинарной классификации изображений парковочных мест
(классы: `free` — свободно, `occupied` — занято). Система должна выводить
состояние места, уверенность модели, историю проверок и краткую статистику,
а также сохранять результаты и формировать отчёты.

## Данные

Используется открытый датасет **ACPDS** (Annotated Car Parking Dataset).
Скрипт [`experiment/create_dataset.py`](experiment/create_dataset.py) вырезает
отдельные парковочные места (ROI) с помощью перспективного преобразования и
формирует разбиение на выборки:

| Выборка | Количество изображений |
|---------|------------------------|
| train   | 4081                   |
| valid   | 687                    |
| test    | 533                    |

Тип разметки — бинарная занятость места (`occupancy_list`).

## Сравниваемые архитектуры

Обучено и сравнено 5 различных архитектур (transfer learning, ImageNet-веса):

| Модель            | Семейство              |
|-------------------|------------------------|
| ResNet-18         | остаточная CNN         |
| EfficientNetV2-M  | композитное масштабирование |
| ConvNeXt-Base     | современная CNN трансформерного типа |
| ViT-B/32          | Vision Transformer     |
| DenseNet-169      | плотно-связная CNN     |

Для каждой модели зафиксированы: метрики качества (accuracy, precision,
recall, F1, confusion matrix), время инференса и размер модели.
Полная таблица сравнения с комментариями формируется скриптом
[`experiment/evaluate_full.py`](experiment/evaluate_full.py) и сохраняется
в `experiment/models_comparison.csv`.

**Лучшая модель:** EfficientNetV2-M (accuracy ≈ 0.987, F1 ≈ 0.983) —
интегрирована в демонстрационное приложение.

## Использование

### Демонстрационное приложение

```bash
cd demo_app
python main.py
```

Возможности приложения:
- загрузка изображения парковочного места;
- вывод состояния (свободно/занято), уверенности и вероятностей классов;
- визуализация внимания модели (Grad-CAM);
- общая статистика по всем проверкам;
- **сохранение истории запусков в `history.json`**;
- **экспорт отчёта в Excel и PDF**, очистка истории.

### Обучение и оценка моделей

```bash
cd experiment

# Подготовка датасета (нужен исходный датасет ACPDS в папке ACPDS/)
python create_dataset.py

# Обучение
# (откройте и выполните train.ipynb)

# Тестирование и таблица метрик
# (откройте и выполните test.ipynb)

# Полное сравнение + анализ ошибок лучшей модели
python evaluate_full.py
```

Результат `evaluate_full.py`:
- `models_comparison.csv` — таблица сравнения 5 моделей (метрики, время,
  размер, комментарий);
- `error_analysis/correct/` — 3 удачных примера лучшей модели;
- `error_analysis/incorrect/` — 3 ошибочных примера с указанием причины.
